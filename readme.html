This tool has only been tested in Chrome. Open the select.html file and enter the location you want to work with. Address and zip codes can be used in the first box. Search using the keyword or/and the type drop down list. Click search for objects and in the list at the bottom you can remove certain results if need be. Clicking the map will move the radius bubble around allowing for additional searches. When done hit send to GEO JSOn and a new window will open with the GEO JSON code.
